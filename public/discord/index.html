Discord Bot(機能一覧)

💬チャット応答

OpenAI API（GPT-4o）を使って、キャラ設定（すんだもん風）に基づく自然な返答を生成。
記憶（memory.json）も毎回考慮され、ユーザーの性格や傾向に合わせた応答をします。
会話履歴（history.json）と記憶をもとに、一貫性のあるキャラ応答が可能。
DiscordのサーバーにキャラBotを招待して、メッセージのやりとりを行います。
PCとスマートフォン、ともにやり取りが可能です。

Discord Bot 機能一覧

🎓 記憶関連コマンド

!覚えて xxx
ユーザー固有の記憶として xxx を保存します。
!記憶一覧
現在保存されている記憶の一覧を番号付きで表示します。
!記憶削除 N
記憶一覧の中から N 番の記憶を削除します（1始まり）。
!忘れて xxx
記憶の中から xxx を含むメモを検索して削除します。

🕒 履歴（会話）関連コマンド

!履歴削除 N
会話履歴を直近 N 件分（ユーザー+Bot のやり取り＝2×N）削除します。
（自動保存）
ユーザーの発言とBotの返答は最大10件分まで履歴として保存されます。

🔒 制限事項

Bot自身の発言は記憶に登録されません。
最大履歴数は10件（ユーザーとBot合わせて10往復）です。
ファイルに書き込めない場合は一時的に記憶や履歴が保存されません。

📁 保存ファイル

memory.json
ユーザーごとの記憶メモを保存
history.json
ユーザーごとの直近の会話履歴を保存
